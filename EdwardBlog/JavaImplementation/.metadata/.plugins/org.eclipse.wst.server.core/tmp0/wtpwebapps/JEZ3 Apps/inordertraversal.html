Traversing a Tree - In-order Traversal

Inorder Traversal is a method of traversing a Binary Tree. In-order traversal works by navigating to the left node, recursively performing the traversal on the left node, printing the active node followed by recursively navigating to the right node. This continues until every node in the tree has been printed/visited. This blog will continue by displaying an interactive tree, showing code in Java which utilizes templates for reuse, and demonstrating unit tests that can be performed on the code.

Java Representation:

/**
 * This action has two template values.  The template value T
 * ensures that any Nodes operations can be supported by
 * any action.  The template value K allows the Action to return any value
 * in the getValue operation.  This should allow the inorder traversal to 
 * be reused for many different types of problems.
 * @param <T>
 * @param <K>
 */
public static interface Action<T extends Node<T>, K>
{
	public void performAction(Node<T> node);
	public boolean complete();
	public K getValue();
}

/**
 * The Node is able to get the left node and the right node.  This is the 
 * requirement for a Binary Tree.  The Node has a template element T which 
 * extends Node<T>.  This allows Action<T extends Node<T>> to use 
 * any public methods inside Node<T> that we do not currently know about
 * and when the traversal goes left or right, it returns the correct Node 
 * type instead of returning a Node that can only go left and right.
 * @param <T>
 */
public static interface Node<T extends Node<T>> 
{
	public T getLeft();
	public T getRight();
}
 
/**
 * This traverses nodes in a inorder fashion.  It recursively calls the left 
 * most node, prints the node and continues by recursively calling the right most node.
**/
public static <T extends BinaryNode<T>, K> K InorderTraversal(T head, 
                                                       Action<T, K> action)
{
	if(head==null || action.complete())
		return action.getValue();
	T left=head.getLeft();
	PreorderTraversal(left, action);
	action.performAction(head);
	T right=head.getRight();
	PreorderTraversal(right, action);
	return action.getValue();
}

Unit Testing

After creating the in-order traversal, it needs to be tested. To do this, we create a tree and ensure the nodes are traversed in the correct order. To ensure the nodes are traversed in the correct order we implement a method to figure out the Kth item of the list. To ensure that the inorder traversal is correct, we create a tree with 3 nodes. A root element "one", and its left element "two" followed by its right element "three". In the intorder traversal, the first element printed is the left element "two", the second printed element is the root element "one" and finally the third node is printed "three". As such, we will show that the tree is traversed in the correct manner for at least three nodes. 

private static class getKthItemAction implements Traversals.Action<TestNode, String>
{
	String value=null;
	int index=0;
	private int desiredIndex;
	private boolean complete;
	public getKthItemAction(int desiredIndex)
	{
		this.desiredIndex=desiredIndex;
		complete=false;
	}
	@Override
	public void performAction(Traversals.Node<TestNode> node) {
		index++;
		if(index==desiredIndex)
		{
			complete=true;
			value=node.toString();
		}
	}

	@Override
	public boolean complete() {
		// TODO Auto-generated method stub
		return complete;
	}

	@Override
	public String getValue() {
		// TODO Auto-generated method stub
		return value;
	}
	
}

private static class TestNode implements Traversals.Node<TestNode>
{
	private TestNode left;
	private TestNode right;
	private String value;
	public TestNode(String value)
	{
		this.value=value;
	}
	public void setLeft(TestNode left)
	{
		this.left=left;
	}
	public void setRight(TestNode right)
	{
		this.right=right;
	}
	@Override
	public TestNode getLeft() {
		// TODO Auto-generated method stub
		return left;
	}

	@Override
	public TestNode getRight() {
		// TODO Auto-generated method stub
		return right;
	}
	@Override
	public String toString()
	{
		return value;
	}


}
public TestNode generateTree()
{
	TestNode one=new TestNode("one");
	TestNode two=new TestNode("two");
	TestNode three=new TestNode("three");
	one.setLeft(two);
	one.setRight(three);
	return one;
}

@Test
public void Inordertest() {
	TestNode root=generateTree();		
	assertTrue("two".equals(Traversals.InorderTraversal(root, new getKthItemAction(1))));
	assertTrue("one".equals(Traversals.InorderTraversal(root, new getKthItemAction(2))));
	assertTrue("three".equals(Traversals.InorderTraversal(root, new getKthItemAction(3))));
}
